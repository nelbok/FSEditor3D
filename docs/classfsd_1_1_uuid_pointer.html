Detailed Description

template<class T>
class fsd::UuidPointer< T >

Pointer to value with Uuid as ID.

Parameters
TAn Entity subclass

Connect two entities by their Uuid instead of their instance.
Use get() to get the instance.
Use uuid() to get the uuid.

Even if the instance is not known at the loading, the pointer will not be destroyed and the uuid will not be updated.
Though, if no instance corresponding to the uuid is found, the pointer will not be valid.
